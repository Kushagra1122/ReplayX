## One-Line Pitch

ReplayX is a Codex-first incident response system that turns a Slack bug report into a live orchestrated diagnosis, reviewed fix path, regression proof plan, postmortem, and reusable incident skill.

## Short Write-Up

ReplayX is built around a simple belief: incident response should feel less like panic and more like playback. We use Codex as the debugging engine behind bounded specialists that analyze incident evidence, compare root-cause hypotheses, challenge weak explanations, rank the safest fix path, and emit reusable incident knowledge. The result is a replayable, inspectable workflow that turns a bug report into engineering-quality artifacts instead of another chat transcript.

## What Judges Should Notice

- ReplayX meaningfully leverages Codex for code-aware debugging work, not just UI copy or chat.
- The system is structured around bounded phases with auditable outputs.
- The dashboard makes the agent workflow easy to understand in under two minutes.
- The product ends with reusable incident memory, not just a one-off recommendation.
